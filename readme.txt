КЛИЕНТ-СЕРВЕРНОЕ ПРИЛОЖЕНИЕ ДЛЯ ОБРАБОТКИ JSON-ФАЙЛОВ (C++)

ПРИНЦИП РАБОТЫ:
Сервер (server) запускается с указанием порта, на котором этот сервер будет запущен (Например: "./server 1234"). После этого сервер ожидает подключения клиента/клиентов.
Клиент (client) запускается с указанием адреса, по которому находится сервер, а также его порта (Например: "./client localhost 1234"). После этого можно отправлять файлы для обработки следующим образом:
"""
<Тип_сортировки> <Имя_файла>
<Тип_сортировки> <Имя_файла>
...
<Тип_сортировки> <Имя_файла>
SEND
"""

Программа предусматривает два вида сортировки: ASC и DESC (Именно в верхнем регистре!). ASC - сортировка по возрастанию, DESC - сортировка по убыванию. После перечисления всех файлов, требующих обработки, следует ввести слово "SEND", которое является "указателем" конца запроса. После этого файлы посылаются на сервер, который сортирует элементы в массиве согласно указанному типу сортировки, и посылает их обратно. Отсортированные файлы будут сохранены в папке PROCESSED_DATA, содержимое файлов будет выведено на экран.

Для завершения работы с сервером следует написать "STOP" в командную строку, и сессия завершится.

ПРИМЕЧАНИЕ: 

ФАЙЛЫ, ТРЕБУЮЩИЕ ОБРАБОТКИ, СЛЕДУЕТ ПОМЕСТИТЬ В ПАПКУ "DATA"!

ДОПОЛНИТЕЛЬНАЯ ИНФОРМАЦИЯ:

Для обработки файлов формата JSON была использована библиотека jsoncpp.
Исполняемые файлы уже скомпилированы, но если требуется скомпилировать их повторно, это можно сделать, введя следующие команды:
"g++ client.cpp some_functions.cpp -Wall -o client -ljsoncpp" - КЛИЕНТ
"g++ server.cpp some_functions.cpp -Wall -o server -ljsoncpp" - СЕРВЕР
